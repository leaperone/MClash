# Security Policy

## Supported versions

Security fixes are provided for the latest published MClash release.

## Reporting a vulnerability

Please use GitHub's **Report a vulnerability** form in the Security tab of
this repository. Reports submitted there are private. Do not open a public
issue for a suspected vulnerability or include subscriptions, proxy
credentials, logs, or backup archives in a public post.

Include the affected MClash version, macOS version, reproduction steps, and
the security impact. Use sanitized test profiles and logs whenever possible.
