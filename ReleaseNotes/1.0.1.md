# MClash 1.0.1

This patch release focuses on local proxy control, responsive layouts, and everyday interaction clarity.

## Improvements

- Added a dedicated Local Proxy editor for HTTP, SOCKS5, and Mixed listeners, with explicit Profile, Custom, and Off modes.
- Port changes are validated before interruption, then restart the core automatically and restore the previous macOS System Proxy state after listener readiness.
- Failed port changes now roll back the runtime configuration, core session, and System Proxy state as one transaction.
- Listener addresses are shown consistently in Overview, Proxies, Settings, and the menu bar; click any visible address to copy it.
- Reorganized Settings into clearer Startup, Routing, Local Proxy, Profiles, Updates, and Advanced sections.
- Improved Proxies and Connections behavior at narrower window sizes so inspectors no longer squeeze the primary list or leave unused workspace.
- Clarified profile activation, proxy health status, topology navigation, and destructive connection confirmations.
- The bundled core executable now uses the product-owned name `mclash-mihomo` inside the application bundle.

## Installation

Download `MClash-1.0.1-macos-arm64.dmg`, open it, and drag MClash to Applications. Existing 1.0.0 installations can also update through **MClash > Check for Updates…**.

This release supports Apple Silicon Macs running macOS 14 or later. Intel Macs are not supported yet.
