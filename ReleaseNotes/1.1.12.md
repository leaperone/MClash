# MClash 1.1.12

This patch makes Traffic and App Routing calmer and more predictable, and adds safe Proxifier rule import.

## Stable workspaces

- Keep the sidebar visible when opening Traffic History by constraining every destination to the finite detail-column size and reducing History's minimum table width.
- Move Traffic's workspace picker, live summary, and actions into a fixed-height header so tabs and controls no longer shift as data changes.
- Keep persistent History controls in one compact fixed-height row; detailed privacy and failure information remains available through help and diagnostics.
- Show global operation errors below each page's fixed header instead of inserting a banner that moves or covers primary controls.

## App Routing clarity

- Replace the large live-data dashboard with a fixed two-row header that shows one concise status and the relevant recovery action.
- Move advanced DNS routing and diagnostic navigation into a compact options menu.
- Keep Rules and Activity tables anchored while Provider state, traffic rates, receipts, and errors change.
- Give the rule editor a fixed viewport and action bar so adding applications or destination groups no longer resizes the sheet.

## Proxifier import

- Import routing rules from Proxifier `.ppx` profiles through a review screen before applying them.
- Preserve rule order, enabled state, multiple application masks, hostname wildcards, IP/CIDR targets, TCP ports, Direct, Block, Proxy, and Chain actions.
- Map Proxy and Chain actions to the current Mihomo profile rules with fail-closed fallback, without importing Proxifier servers, addresses, chains, usernames, or passwords.
- Skip redundant Direct defaults, require explicit selection for catch-all proxy rules, and reject DTD/entity-based XML.
- Compile large hostname-mask groups once per configuration revision into lookup indexes so imported lists with thousands of targets do not rebuild or linearly scan the whole list for every flow.

## Verification

- Added Proxifier parser, conversion, security, duplicate-name, catch-all, multi-application, hostname-glob, and 3,000-target indexing coverage.
- Verified the provided 30-rule Proxifier profile through the opt-in read-only integration fixture.
- Verified 258 application tests and 101 shared networking tests, followed by warnings-as-errors direct links of the host and Network Extension.
