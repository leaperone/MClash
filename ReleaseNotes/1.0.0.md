# MClash 1.0.0

The first public release of MClash: a native macOS controller for the mihomo Alpha core.

## Highlights

- Native SwiftUI app and menu bar controls for Rule, Global, and Direct routing.
- Responsive proxy workspace with ordered nested groups, node search, latency tests, topology, and route inspection.
- Local and remote profile management with validation, transactional activation, backup, and rollback.
- Live connections, traffic, rules, providers, and diagnostic logs.
- Safe macOS HTTP, HTTPS, and SOCKS proxy activation with complete restoration on disconnect or quit.
- Signed in-app updates delivered through Sparkle and GitHub Releases.

## Installation

1. Download `MClash-1.0.0-macos-arm64.dmg`.
2. Open the disk image and drag MClash to Applications.
3. Open MClash from Applications.

This release supports Apple Silicon Macs running macOS 14 or later. Intel Macs are not supported yet.

MClash bundles a pinned, checksum-verified mihomo Alpha binary. The bundled core remains a separate process and retains its upstream license and notices inside the application bundle.

The Release also includes the complete source tree for the exact bundled mihomo revision and Sparkle's MIT license notice.
