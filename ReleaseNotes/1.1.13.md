# MClash 1.1.13

This patch makes Proxifier profile files selectable in the App Routing importer.

## Proxifier import

- Register the `.ppx` filename extension as an imported Proxifier Profile type in the signed app bundle.
- Use the same declared type in the macOS file importer so `.ppx` files are enabled instead of appearing greyed out.
- Add release-packaging coverage to prevent the bundle declaration and importer type from drifting apart again.

## Verification

- Verified the provided `.ppx` profile remains fully parseable after the file-type fix.
- Verified 259 application tests and 101 shared networking tests, followed by warnings-as-errors direct links of the host and Network Extension.
