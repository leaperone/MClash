# MClash 1.0.2

This patch release removes mihomo's first-run dependency on external GEO database downloads.

## Improvements

- Every release now fetches a fresh, immutable snapshot from MetaCubeX/meta-rules-dat's release branch and verifies the upstream SHA-256 files before packaging.
- Bundles `geoip.metadb`, `GeoIP.dat`, `GeoSite.dat`, and `ASN.mmdb` to cover default MMDB, GeoData, GEOSITE, and IP-ASN profile modes.
- Seeds both the profile-validation home and the live core home before mihomo starts, so importing or connecting a profile does not need GitHub access for GEO rules.
- Preserves existing non-empty GEO files, allowing mihomo or the user to keep newer database updates.
- Fails closed if the bundled release snapshot is incomplete or fails its integrity manifest.

## Installation

Download `MClash-1.0.2-macos-arm64.dmg`, open it, and drag MClash to Applications. Existing installations can update through **MClash > Check for Updates…**.

This release supports Apple Silicon Macs running macOS 14 or later. Intel Macs are not supported yet.
