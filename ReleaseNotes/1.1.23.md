# MClash 1.1.23

This patch restores the useful menu-bar controls, keeps the status item visually stable, and removes redundant local-router DNS connections from Mihomo.

## Menu Bar

- Use a compact monochrome network symbol in Network Icon mode instead of scaling the full application icon into the macOS menu bar.
- Show only download, upload, and connection count in Proxy Status mode, with smaller type, fixed-width fields, and short `B/K/M/G`-style units that do not shift the status item as values change.
- Preserve the same three-field width while disconnected or while live telemetry is unavailable.
- Restore Routing Mode, Quick Routes, local proxy addresses, macOS System Proxy, and the main management shortcuts in the menu popover.
- Tighten the popover typography, spacing, and metric cards while keeping recovery guidance and mutually exclusive routing controls clear.

## DNS Routing

- Relay local and private DNS resolvers directly from the DNS Network Extension instead of sending endpoints such as `192.168.1.1:53` through Mihomo's private SOCKS listener.
- Keep public DNS resolvers on Profile routing and leave ordinary LAN application routing rules unchanged.
- Add regression coverage for RFC 1918, IPv6 ULA, public resolver routing, compact status formatting, and trusted MClash DNS egress.
